# Social Media Data Analysis using Power BI

## Project Overview
This project analyzes social media engagement, platform performance, brand impact, campaign effectiveness, and user sentiment using an interactive Power BI dashboard. The objective is to help marketing teams understand content performance and user behavior for better decision-making.

## Dataset
The dataset contains social media engagement data across multiple platforms, brands, campaigns, and topic categories, including:
- Engagement metrics
- Platform information
- Brand & campaign data
- Sentiment and emotion labels
- Time-based activity data

## Tools & Technologies
- Power BI
- Microsoft Excel
- DAX
- Data Modeling
- Data Visualization

## Dashboard Features

### 1. Executive Overview
- Total Engagement: 48M+
- Engagement YTD and YoY Growth
- Monthly Engagement Trend
- Platform-wise and Topic-wise Engagement

### 2. Platform Performance
- Engagement by Platform
- Engagement Rate vs Volume
- Sentiment distribution per platform
- Platform engagement trend

### 3. Brand & Campaign Performance
- Top Brand and Best Campaign
- Campaign phase impact
- Campaign performance matrix
- Brand engagement trends

### 4. User Behaviour & Sentiment Analysis
- Sentiment distribution
- Emotion vs Engagement
- Topic category vs sentiment
- User activity trends

## Key Insights
- YouTube is the top-performing platform by engagement.
- Samsung is the most engaging brand.
- NextGeneration is the best performing campaign.
- Positive and negative sentiments are almost equally distributed.
- Marketing and Pricing topics generate the highest engagement.
